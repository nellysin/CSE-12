Lab 3: ASCII-risks (Asterisks)

------------
DESCRIPTION

In this lab, the user will enter a number that is greater than 0 and it will run through several nested loops to print a variable-sized ASCII diamonds, "*", and a sequence of embedded numbers.

------------
FILES

-
Lab3.asm

This file is the program or assembly code for the lab.

------------
INSTRUCTIONS

This program is intended to run using the MIPS Assembler and Runtime Simulator, MARS. The program will ask for a number greater than 0 and the user must enter a number for it to execute.
